Architecture CI/CD de ce Portfolio

Ce portfolio est déployé automatiquement via une pipeline CI/CD complète — de la création d'un tag Git jusqu'à la mise en production sur un VPS cloud avec Docker et Nginx.

Site servi en production sur port 80

Versioning par tags Git

Chaque release est taguée (v1.0.0, v1.1.0...) — rollback possible en 30 secondes

Containerisation Docker

Image Nginx Alpine légère (<50MB), reproductible sur n'importe quel environnement

Déploiement sécurisé SSH

Rollback instantané

Health Checks
