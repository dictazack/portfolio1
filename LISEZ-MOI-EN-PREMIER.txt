## 🔄 ARCHITECTURE CI/CD

```
┌─────────────┐    Push/PR    ┌──────────────────┐    Build    ┌─────────────────┐
│   GitHub    │──────────────▶│ GitHub Actions   │────────────▶│  GitHub Pages   │
│  Repository │               │ (Auto Deploy)    │             │ (Live Website)  │
└─────────────┘               └──────────────────┘             └─────────────────┘
```

**Avantages:**
- ✅ Déploiement automatique à chaque modification
- ✅ Aucun serveur à gérer
- ✅ HTTPS automatique
- ✅ CDN global inclus
- ✅ 100% gratuit
